### <u>Run Locally</u>

Clone the project

https://github.com/rahulr41180/rosh-assignment

### <u>Run Commands on Frontend Side</u>
- Go Into Client Directory
- cd client
- npm install
- npm start

It will open at "http://localhost:3000"

### <u>Run Commands on Server Slide</u>
- Go Into Server Folder Directory
- npm install
- npm server
